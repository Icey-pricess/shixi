# 使用TypeScript+vue3构建博客网站

### 项目简介

- Vue3+TypeScript开发，使用新版 Vuex 和 Vue-Router 全家桶完成。
- 从0到1，不借用UI矿建，手动开发通用组件，如Modal，Loader，Upload，Form，DropDown
- 自定义指令v-click-outside封装
- 实现内容包含登录注册，专栏列表，专栏详情，文章列表，文章详情，编辑删除文章，更改User信息等

### 安装依赖
```
npm install
```

### 运行本地开发环境
```
npm run serve
```

### 打包构建
```
npm run build
```
